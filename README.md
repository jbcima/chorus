## Quick start

1.  **Develop locally**

    Start up the site ans then navigate to http://localhost:8000

    ```shell
    gatsby develop
    ```

2.  **Build and Deploy**

    Run the commands to build the site and deploy it online. 

    ```shell
    npm run build
    npm run deploy 
    ```
